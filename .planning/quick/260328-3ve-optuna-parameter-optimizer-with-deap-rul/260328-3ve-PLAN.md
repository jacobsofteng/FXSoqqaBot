Create function `async def evolve_weights(base_settings: BotSettings, bt_config: BacktestConfig, n_generations: int = 10, population_size: int = 20, seed: int = 42) -> dict[str, float]`:
1. Create DEAP toolbox with: `initIterate` for Individual creation (3 genes), `cxBlend(alpha=0.5)`, `mutGaussian(mu=0, sigma=0.1, indpb=0.3)`, `selTournament(tournsize=3)`.
2. Initialize population of `population_size` individuals, each with 3 random floats in their respective bounds. Set `random.seed(seed)` first.
3. For each generation:

   b. Clamp genes to their respective bounds.
   c. Wait -- since evolve_weights is itself async, use `await validator.run_walk_forward()` directly. The fitness is `(wf_result.aggregate_profit_factor,)`.
   d. Select next generation.
   e. Print progress: `f"  DEAP Gen {gen}/{n_generations}: best_fitness={best:.4f}"`.
4. Return best individual as `dict[str, float]` mapping WEIGHT_NAMES to values.

**5. Create `src/fxsoqqabot/optimization/optimizer.py`:**

This orchestrates the full two-phase optimization.

Create function `def run_optimization(settings: BotSettings, bt_config: BacktestConfig, n_trials: int = 50, n_generations: int = 10, output_path: str = "config/optimized.toml", skip_ingestion: bool = False) -> None`:

This is a SYNCHRONOUS function. Optuna drives the outer loop, each trial objective calls `asyncio.run()` to bridge to async BacktestEngine.

**Phase A: Optuna TPE**
1. If not skip_ingestion, run ingestion: `loader = HistoricalDataLoader(bt_config); loader.ingest_all()`.
2. Create Optuna study: `optuna.create_study(direction="maximize", sampler=TPESampler(seed=42), study_name="fxsoqqabot-optimize")`.
3. Define objective function:
   ```python
   def objective(trial: optuna.Trial) -> float:
       params = sample_trial(trial)
       trial_settings = apply_params_to_settings(settings, params)
       wf_result = asyncio.run(_async_walk_forward(trial_settings, bt_config))
       return min(wf_result.aggregate_profit_factor, 10.0)  # Cap at 10.0
   ```
   Where `_async_walk_forward` creates BacktestEngine + HistoricalDataLoader + WalkForwardValidator and calls `run_walk_forward()`.
4. Run: `study.optimize(objective, n_trials=n_trials, show_progress_bar=True)`.
5. Print best trial summary: trial number, value, params.

**Phase B: DEAP GA (signal weights)**
6. Build settings with best Optuna params frozen: `optuna_settings = apply_params_to_settings(settings, study.best_trial.params)`.
7. Run `best_weights = asyncio.run(evolve_weights(optuna_settings, bt_config, n_generations=n_generations))`.
8. Merge best Optuna params + best DEAP weights into `final_params` dict.
9. Build final settings: `final_settings = apply_params_to_settings(optuna_settings, best_weights)`.

**Final Validation**
10. Run full validation:
    ```python
    async def _validate_final(final_settings, bt_config):
        loader = HistoricalDataLoader(bt_config)
        engine = BacktestEngine(final_settings, bt_config)
        validator = WalkForwardValidator(engine, loader, bt_config)
        wf = await validator.run_walk_forward()
        oos = await validator.evaluate_oos(wf)
        # Collect validation trade PnLs for Monte Carlo
        pnls = [t.pnl for w in wf.windows for t in w.val_result.trades]
        mc = run_monte_carlo(np.array(pnls), bt_config.starting_equity, bt_config.n_monte_carlo, bt_config.mc_max_drawdown_pct, bt_config.mc_seed) if pnls else None
        return wf, oos, mc
    ```
    Run via `asyncio.run(_validate_final(...))`.
11. Print formatted validation results (reuse formatting pattern from runner.py -- profit factor, OOS ratio, MC pass/fail).

**TOML Output**
12. Only write if ALL gates pass (wf.passes_threshold AND oos.passes_threshold AND (mc is None or mc.passes_threshold)):

    import tomli_w
