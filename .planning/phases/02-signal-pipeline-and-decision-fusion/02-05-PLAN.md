    ```python
    @dataclass(frozen=True, slots=True)
    class FusionResult:
        direction: float           # -1.0 to +1.0 (final trade direction)
        composite_score: float     # Fused confidence-weighted score
        fused_confidence: float    # Overall fusion confidence 0-1
        should_trade: bool         # Exceeds threshold for current phase
        regime: RegimeState        # From chaos module
        module_scores: dict        # Individual module contributions for debugging
        confidence_threshold: float  # Active threshold for this decision
    ```

    ```python
    class FusionCore:
        def __init__(self, config: FusionConfig):
            self._config = config
            self._logger = structlog.get_logger().bind(component="fusion_core")

        def fuse(
            self,
            signals: list[SignalOutput],
            weights: dict[str, float],
            confidence_threshold: float,
        ) -> FusionResult:
    ```
    - Fusion formula per D-01: for each signal, weighted_score = signal.direction * signal.confidence * weights[signal.module_name]
    - Composite = sum(weighted_scores) / sum(signal.confidence * weight) -- normalized by total confidence weight
    - If total_weight == 0: composite = 0.0
    - fused_confidence = sum(signal.confidence * weight) / len(signals) if signals else 0.0
    - should_trade = abs(composite) > 0 and fused_confidence >= confidence_threshold
    - direction = sign(composite)
    - Extract regime from chaos module signal (first signal with regime != None), default RANGING
    - Per D-05: weights adapt from accuracy only, NOT from regime. No regime-to-weight mapping anywhere in this code.
    - Per D-07: no hardcoded ranging behavior. If fusion says trade in a range, trade.

      - EMA update: accuracy = alpha * correct + (1 - alpha) * old_accuracy per D-02

      - Normalize accuracies so they sum to 1.0

      - If all accuracies zero: return equal weights

    - `load_state(self, state: dict) -> None`: Restore from serialized state

    **trade_manager.py** (FUSE-05, D-08, D-09, D-10, D-11):
    ```python
    @dataclass(frozen=True, slots=True)
    class TradeDecision:
        action: str               # "buy", "sell", "hold", "tighten_sl"
        sl_distance: float        # ATR-based, regime-adjusted
        tp_distance: float        # SL * RR ratio
        lot_size: float           # From PositionSizer, adjusted for regime
        confidence: float         # From FusionResult
        regime: RegimeState       # Current regime
        reason: str               # Human-readable reason for the decision

    class TradeManager:
        def __init__(
            self,
            fusion_config: FusionConfig,
            phase_behavior: PhaseBehavior,
            order_manager: OrderManager | None,  # None in testing
            position_sizer: PositionSizer,
            breaker_manager: CircuitBreakerManager | None,  # None in testing
        ):
            self._config = fusion_config
            self._phase = phase_behavior
            self._orders = order_manager
            self._sizer = position_sizer
            self._breakers = breaker_manager
            self._open_position_ticket: int | None = None
            self._open_position_entry: float = 0.0
            self._open_position_regime: RegimeState | None = None
            self._logger = structlog.get_logger().bind(component="trade_manager")
    ```

    - `def compute_sl_tp(self, atr: float, regime: RegimeState, direction: float, current_price: float) -> tuple[float, float, float, float]` (D-09):
      - sl_distance = atr * config.sl_atr_base_multiplier
      - Regime adjustments: adjustments = phase_behavior.get_regime_adjustments(regime)
      - If "sl_widen_factor" in adjustments: sl_distance *= adjustments["sl_widen_factor"]
      - rr_ratio = phase_behavior.get_rr_ratio(regime)
      - tp_distance = sl_distance * rr_ratio
      - If direction > 0 (buy): sl_price = current_price - sl_distance, tp_price = current_price + tp_distance
      - If direction < 0 (sell): sl_price = current_price + sl_distance, tp_price = current_price - tp_distance
      - Return (sl_price, tp_price, sl_distance, tp_distance)

    - `async def evaluate_and_execute(self, fusion_result: FusionResult, equity: float, current_price: float, atr: float) -> TradeDecision`:
      1. Check position limit (D-11): if self._open_position_ticket is not None:
         - Check for adverse regime transition (D-08): if fusion_result.regime differs from _open_position_regime AND is HIGH_CHAOS/PRE_BIFURCATION:
           - Tighten SL: new_sl = entry + (current - entry) * 0.5 (lock in 50% of profit, or reduce 50% of loss)
           - If self._orders: await self._orders.modify_sl(ticket, new_sl) -- fire and forget
           - Return TradeDecision(action="tighten_sl", ...)
         - Otherwise: return TradeDecision(action="hold", ..., reason="position already open per D-11")
      2. If not fusion_result.should_trade: return TradeDecision(action="hold", ..., reason="below confidence threshold")
      3. Check breakers: if self._breakers and not self._breakers.can_trade(): return TradeDecision(action="hold", ..., reason="circuit breaker tripped")
      4. Compute SL/TP: sl_price, tp_price, sl_dist, tp_dist = self.compute_sl_tp(atr, fusion_result.regime, fusion_result.direction, current_price)
      5. Position sizing:
         - sizing = self._sizer.calculate_lot_size(equity, sl_dist)
         - Apply regime adjustment: if adjustments has "size_reduction": sizing_lot *= (1 - adjustments["size_reduction"])
         - If not sizing.can_trade: return TradeDecision(action="hold", ..., reason=sizing.skip_reason)
      6. Execute trade:
         - action = "buy" if fusion_result.direction > 0 else "sell"
         - If self._orders: fill = await self._orders.place_market_order(symbol="XAUUSD", action=action, volume=lot_size, sl=sl_price, tp=tp_price)
         - If fill: record _open_position_ticket, _open_position_entry, _open_position_regime
         - Return TradeDecision(action=action, sl_distance=sl_dist, tp_distance=tp_dist, lot_size=lot_size, confidence=fusion_result.fused_confidence, regime=fusion_result.regime, reason="fusion threshold exceeded")

    - `def record_position_closed(self, ticket: int) -> None`: Clear _open_position_ticket when position closes (called by engine on POSITION_CLOSED event)

    - `def get_trailing_params(self, regime: RegimeState) -> dict | None`: Delegate to phase_behavior.get_trailing_stop_params(regime)
