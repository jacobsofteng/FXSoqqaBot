class RegimeState(str, Enum):

    - `compute_lyapunov(close_prices: np.ndarray, emb_dim: int = 10, min_length: int = 300) -> tuple[float, float]`
    - Use `nolds.lyap_r(close_prices, emb_dim=emb_dim, fit="RANSAC")` -- Rosenstein algorithm
    - Return (0.0, 0.0) if len < min_length
    - Confidence = min(1.0, len(close_prices) / (min_length * 3))
    - Positive = chaotic/unstable, Negative = stable, ~0 = neutral
    - Wrap in try/except: return (0.0, 0.0) on error

    **fractal.py** (CHAOS-03):
    - `compute_fractal_dimension(close_prices: np.ndarray, emb_dim: int = 10, min_length: int = 200) -> tuple[float, float]`
    - Use `nolds.corr_dim(close_prices, emb_dim=emb_dim)` -- Grassberger-Procaccia
    - Return (1.5, 0.0) if len < min_length (default midrange assumption)
    - Confidence = min(1.0, len(close_prices) / 600.0)
    - Clamp output to [1.0, 2.0] range for 1D time series
    - Wrap in try/except: return (1.5, 0.0) on error

    **feigenbaum.py** (CHAOS-04):
    - `detect_bifurcation_proximity(close_prices: np.ndarray, order: int = 5) -> tuple[float, float]`
    - Use `scipy.signal.argrelextrema(close_prices, np.greater, order=order)` to find peaks
    - Compute intervals between successive peaks with `np.diff`
    - Compute ratios of successive interval differences
    - proximity = 1.0 - min(1.0, abs(avg_ratio - 4.669201609) / 4.669201609) where 4.669201609 is Feigenbaum delta
    - Return (0.0, 0.0) if < 50 points or < 4 peaks or < 1 ratio computed
    - Confidence = min(1.0, len(ratios) / 10.0)
    - This is novel territory with LOW confidence -- keep implementation simple per research guidance

    **entropy.py** (CHAOS-05):

    - Bin returns into histogram with `np.histogram(returns, bins=bins, density=True)`
    - Use `scipy.stats.entropy(probabilities)` for Shannon entropy on the distribution
    - Normalize: divide by max possible entropy `np.log(bins)`
    - Return (0.5, 0.0) if < min_length -- mid-range assumption
    - Confidence = min(1.0, len(close_prices) / 500.0)
    - High entropy = crowd uncertainty/panic; Low entropy = orderly/trending
