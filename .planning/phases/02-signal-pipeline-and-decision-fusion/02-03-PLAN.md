    **volume_delta.py** (FLOW-01):
    - `compute_volume_delta(bid: np.ndarray, ask: np.ndarray, last: np.ndarray, volume_real: np.ndarray, window: int = 100) -> tuple[float, float, float, float]`
    - Classify ticks as buy-initiated if `last >= ask` (lifting the offer), sell-initiated if `last <= bid` (hitting the bid). Ticks between bid and ask are ambiguous per Research Pitfall 3.
    - Use last `window` ticks only: slice all arrays to `[-window:]`
    - Return (cumulative_delta, buy_volume, sell_volume, ambiguous_pct)
    - ambiguous_pct = count of ambiguous ticks / total ticks -- used to reduce confidence when > 30%
    - Empty arrays: return (0.0, 0.0, 0.0, 1.0)

    **aggression.py** (FLOW-02):
    - `compute_aggression_imbalance(bid: np.ndarray, ask: np.ndarray, last: np.ndarray, volume_real: np.ndarray, window: int = 200) -> tuple[float, float, float]`
    - For last `window` ticks, compute volume hitting ask (buy aggression) vs volume hitting bid (sell aggression)
    - Imbalance ratio = (buy_vol - sell_vol) / (buy_vol + sell_vol) if total > 0 else 0.0. Range: [-1, +1]
    - Z-score: compute rolling mean and std of imbalance over the window. If std == 0, z-score = 0.0
    - Confidence = min(1.0, abs(zscore) / 3.0) -- higher z-score = more significant = more confident
    - Return (imbalance_ratio, zscore, confidence)

    Also add to this file (FLOW-05):
    - `detect_hft_signatures(time_msc: np.ndarray, spread: np.ndarray, volume_real: np.ndarray, tick_velocity_threshold: float = 5.0, spread_widen_multiplier: float = 2.0) -> tuple[bool, float]`
    - Compute tick velocity: count ticks per second using time_msc differences
    - Compute mean tick velocity and std over the window
    - HFT signature = tick velocity > mean + tick_velocity_threshold * std AND spread > mean_spread * spread_widen_multiplier simultaneously
    - Return (is_hft_detected, confidence) where confidence = 0.0-1.0 based on how far above threshold

    **institutional.py** (FLOW-04, per D-14):
    - `detect_institutional_footprints(bid: np.ndarray, ask: np.ndarray, last: np.ndarray, volume_real: np.ndarray, spread: np.ndarray, time_msc: np.ndarray, volume_threshold: float = 3.0, price_tolerance: float = 0.5, min_repeats: int = 3) -> tuple[float, float, dict]`
    - **Absorption detection**: Find ticks where volume > mean + volume_threshold * std BUT price change is < 0.1 * ATR (large volume, no movement). Score = count / expected.
    - **Iceberg reload detection**: Group ticks by price level (within price_tolerance). If any level has >= min_repeats large-volume ticks, flag as iceberg. Score = max(repeats / (2 * min_repeats), 1.0).
    - **Volume profile clustering**: Compute volume-at-price profile. Find high-volume nodes (peaks). These are institutional levels.
    - Combine: institutional_score = weighted average of (absorption_score * 0.4 + iceberg_score * 0.3 + volume_profile_score * 0.3)
    - Direction: positive if institutional buying detected (delta at institutional levels > 0), negative if selling
    - Return (institutional_score -1 to +1, confidence 0-1, signals_dict with absorption/iceberg/profile details)
    - Empty arrays: return (0.0, 0.0, {})

    **dom_quality.py** (per D-15):
    - `class DOMQualityChecker`:
      - `__init__(self, config: FlowConfig)`: store config, init tracking state
      - `_snapshots: collections.deque` of (timestamp, depth_count) tuples, maxlen based on duration
      - `_dom_enabled: bool = False`

      - `_evaluate_quality(self) -> None`: Check if depth >= config.dom_min_depth on both bid and ask sides in recent snapshots, and update rate >= config.dom_min_update_rate per second. Set `_dom_enabled` accordingly. Log state changes.

      - `needs_recheck(self) -> bool`: True if dom_recheck_interval_minutes elapsed since last check

    **dom_analyzer.py** (FLOW-03):
    - `analyze_dom(dom: DOMSnapshot | None, min_depth: int = 5) -> tuple[float, float]`
    - If dom is None or entries is empty tuple, return (0.0, 0.0)
    - Separate entries by type: type=1 is sell (ask side), type=2 is buy (bid side) per DOMEntry.type
    - Compute bid_volume = sum of volume_dbl for buy entries, ask_volume = sum for sell entries
    - Imbalance = (bid_volume - ask_volume) / (bid_volume + ask_volume) if total > 0 else 0.0
    - Check for large hidden orders: any single entry > 3x average entry volume
    - Confidence = min(1.0, (len(buy_entries) + len(sell_entries)) / (2 * min_depth))
    - Return (imbalance_direction, confidence)

    **module.py** (FLOW-06, D-13):
    - `class OrderFlowModule` implementing SignalModule Protocol
    - Constructor takes `config: FlowConfig`
    - Private `_dom_checker: DOMQualityChecker`
    - `name` property returns `"flow"`
    - `async def initialize(self) -> None`: initialize DOMQualityChecker
    - `async def update(self, tick_arrays, bar_arrays, dom) -> SignalOutput`:
      1. Extract tick arrays: bid, ask, last, volume_real, spread, time_msc
      2. If tick arrays are empty (bid has len 0), return neutral: SignalOutput(module_name="flow", direction=0.0, confidence=0.0)
      3. Compute volume delta: `compute_volume_delta(bid, ask, last, volume_real, config.volume_delta_window)`
      4. Compute aggression: `compute_aggression_imbalance(bid, ask, last, volume_real, config.aggression_window)`
      5. Detect HFT: `detect_hft_signatures(time_msc, spread, volume_real, config.hft_tick_velocity_threshold, config.hft_spread_widen_multiplier)`
      6. Detect institutional: `detect_institutional_footprints(bid, ask, last, volume_real, spread, time_msc, config.institutional_volume_threshold, config.institutional_price_tolerance, config.institutional_min_repeats)`
      7. DOM analysis (optional per D-13):
         - If dom is not None: `_dom_checker.record_snapshot(dom)`
         - If `_dom_checker.is_dom_enabled`: dom_imbalance, dom_conf = `analyze_dom(dom, config.dom_min_depth)`
         - Else: dom_imbalance, dom_conf = 0.0, 0.0
      8. Combine signals (~80% tick / ~20% DOM per D-13):
         - Normalize volume delta to [-1, +1] range: `np.clip(cum_delta / (buy_vol + sell_vol + 1e-10), -1, 1)`
         - tick_direction = 0.6 * normalized_delta + 0.2 * aggression_imbalance + 0.2 * institutional_score
         - If dom_conf > 0: direction = 0.8 * tick_direction + 0.2 * dom_imbalance. Else: direction = tick_direction
         - Overall confidence = weighted average of component confidences, reduced by ambiguous_pct if > 0.3
         - If HFT detected, add HFT flag to metadata and slightly reduce confidence (HFT noise)
      9. Return SignalOutput(module_name="flow", direction=np.clip(direction, -1, 1), confidence=confidence, metadata={...})
    - Use structlog with `.bind(component="flow_module")` for logging
