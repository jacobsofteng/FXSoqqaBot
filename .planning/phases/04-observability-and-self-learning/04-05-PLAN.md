    **Frozen dataclass `ShadowVariant`:**
    NOT frozen -- needs mutable state for trade tracking.
    ```python
    @dataclass
    class ShadowVariant:
        variant_id: str
        mutated_params: dict[str, float]
        paper_executor: PaperExecutor
        trade_results: list[dict] = field(default_factory=list)
        created_at: datetime = field(default_factory=lambda: datetime.now(UTC))

        @property
        def trade_count(self) -> int:
            return len(self.trade_results)

        @property
        def fitness_score(self) -> float:
            if not self.trade_results:
                return 0.0
            pnls = [t.get("pnl", 0.0) for t in self.trade_results if t.get("pnl") is not None]
            if not pnls:
                return 0.0
            wins = sum(1 for p in pnls if p > 0)
            return wins / len(pnls) if pnls else 0.0
    ```

    **Class ShadowManager:**
    - `__init__(self, config: LearningConfig, starting_balance: float = 20.0)`:
      Create `config.n_shadow_variants` ShadowVariant instances. Each gets:
      - Unique `variant_id` like `"shadow_0"`, `"shadow_1"`, etc.
      - `mutated_params` from `_generate_mutations()`
      - Own `PaperExecutor(starting_balance=starting_balance)` -- per anti-pattern warning, NEVER shared.

    - `_generate_mutations(self) -> dict[str, float]`:
      For each param in PARAM_NAMES, generate a value: take a base value (center of bounds), apply Gaussian perturbation (sigma=0.1 * range), clamp to bounds. Return as dict.

    - `record_variant_trade(self, variant_id: str, trade_result: dict) -> None`:
      Append trade_result to the variant's trade_results list.

    - `evaluate_promotion(self, variant: ShadowVariant, live_trades: list[dict], equity: float) -> dict`:
      Returns `{"should_promote": bool, "p_value": float, "variant_fitness": float, "live_fitness": float, "reason": str}`.

      Steps:
      1. If variant.trade_count < config.min_promotion_trades: return should_promote=False, reason="insufficient trades".
      2. Extract pnl lists from variant.trade_results and live_trades.
      3. Run Mann-Whitney U test: `stats.mannwhitneyu(variant_pnls, live_pnls, alternative="greater")`.
      4. If p_value >= config.promotion_alpha: return should_promote=False, reason=f"p={p_value:.4f} >= {config.promotion_alpha}".
      5. If passes: return should_promote=True. (Walk-forward validation is called separately by the orchestrator in engine integration.)

    - `promote_variant(self, variant_id: str) -> dict[str, float]`:
      Get the variant's mutated_params. Reset the variant (new PaperExecutor, new mutations, clear trade_results). Return the promoted params dict. Log: "Variant promoted", variant_id=variant_id.

    - `reset_variant(self, variant_id: str) -> None`:
      Replace variant with fresh mutations and new PaperExecutor.

    - `get_variant_status(self) -> list[dict]`:
      Return list of dicts with variant_id, trade_count, fitness_score, age (seconds since created_at), mutated_params for all variants.

    - `get_variants(self) -> list[ShadowVariant]`: Return the variants list.

    Create `tests/test_shadow.py` with tests. Use small n_shadow_variants=3. Mock trade results with known P&L distributions to test statistical promotion.
