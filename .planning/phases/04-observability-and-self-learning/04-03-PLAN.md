       ```python
       from __future__ import annotations
       import asyncio
       from pathlib import Path
       ```

       Class `DashboardServer`:
       - `__init__(self, config: WebConfig, state: TradingEngineState, trade_logger=None, kill_callback=None, pause_callback=None)`:
         Create FastAPI app instance with title="FXSoqqaBot Dashboard".
         Mount static files: `app.mount("/static", StaticFiles(directory=str(Path(__file__).parent / "static")), name="static")`.
         Store config, state, trade_logger references.
         Register routes.

       - Routes to register:
         - `GET /` -> serve index.html via FileResponse
         - `GET /api/trades` with query params: regime, outcome, start_date, end_date, min_confidence (all optional), limit=100 -> delegates to trade_logger.query_trades() if trade_logger set, else returns []
         - `GET /api/equity` -> returns equity_history from state as `{"data": [...]}` (for Plotly)
         - `GET /api/regime-timeline` -> returns recent regime states from trade_logger (query trades and extract regime + timestamp pairs)
         - `GET /api/module-weights` -> returns module weight history (placeholder data until learning loop wired)
         - `POST /api/kill?api_key={key}` -> validates api_key against config.api_key. If mismatch: raise HTTPException(403, "Invalid API key"). If match: call kill_callback, return {"status": "killed"}.
         - `POST /api/pause?api_key={key}` -> validate key, call pause_callback, return {"status": "paused"/"resumed"} toggling state.is_paused.
         - `WebSocket /ws/live` -> accept connection, loop sending state.to_dict() every 1 second. Handle WebSocketDisconnect gracefully. Log connections/disconnections.

       - `get_app(self) -> FastAPI`: Returns the FastAPI app instance for uvicorn.

       - `async def start(self) -> None`: Start uvicorn programmatically:
         ```python
         import uvicorn
         config = uvicorn.Config(self._app, host=self._config.host, port=self._config.port, log_level="warning")
         self._server = uvicorn.Server(config)
         await self._server.serve()
         ```

       - `async def stop(self) -> None`: Trigger server shutdown if running.
