    **Class LearningLoopManager:**
    - `__init__(self, config: LearningConfig, trade_logger: TradeContextLogger, equity: float = 20.0)`:
      Initialize all sub-components:
      - `self._evolution = EvolutionManager(config)`
      - `self._shadow = ShadowManager(config, starting_balance=equity)`
      - `self._classifier = RegimeClassifier()`
      - `self._retirement = RuleRetirementTracker(rule_names=PARAM_NAMES, alpha=0.1, min_trades=config.retirement_min_trades, retirement_threshold=config.retirement_threshold)`
      - `self._analyzer = SignalAnalyzer()`
      - `self._trade_logger = trade_logger`
      - `self._config = config`
      - `self._trades_since_evolve = 0`
      - `self._trades_since_retrain = 0`
      - `self._total_trades = 0`
      - `self._current_equity = equity`
      - `self._last_mutation_events: list[dict] = []`

    - `async def on_trade_closed(self, trade_result: dict) -> list[dict]`:
      Main entry point called when a live trade closes. Returns list of mutation events (for TUI display).
      1. Increment counters.
      2. Update equity from trade_result.
      3. Record outcome for retirement tracker: determine which rules contributed, record success based on pnl > 0.
      4. Record for shadow variants: feed same trade context to all variants (they already have independent PaperExecutors processing market data).
      5. If `_trades_since_evolve >= config.evolve_every_n_trades`:
         - `await asyncio.to_thread(self._run_evolution)` (blocking DEAP in thread per Pattern 3)
         - Reset counter.
      6. If `_trades_since_retrain >= 100`:
         - `await asyncio.to_thread(self._retrain_classifier)`
         - Reset counter.
      7. Check promotions: `mutations = self._check_promotions()`
      8. Return mutation events.

    - `def _run_evolution(self) -> None`:
      Blocking function -- runs in thread.
      1. Get recent trades from trade_logger: `trades = self._trade_logger.query_trades(limit=200)`
      2. `result = self._evolution.run_generation(trades, self._current_equity)`
      3. Log: "Evolution generation complete", generation=result["generation"], best_fitness=result["best_fitness"].
      4. Update shadow variant mutations from evolution best individuals.

    - `def _retrain_classifier(self) -> None`:
      Blocking function -- runs in thread.
      1. Get all trades: `trades = self._trade_logger.query_trades(limit=1000)`
      2. `result = self._classifier.train(trades)`
      3. Log: "Classifier retrained", is_trained=result["is_trained"], accuracy=result.get("cv_accuracy", 0).

    - `def _check_promotions(self) -> list[dict]`:
      For each shadow variant:
      1. Get live trades from trade_logger.
      2. Evaluate promotion.
      3. If should_promote:
         - promoted_params = shadow.promote_variant(variant_id)
         - Create mutation event dict: {"mutation": True, "param": "multiple", "old": "live_params", "new": str(promoted_params), "reason": f"variant promoted p={p_value:.4f}"}
         - Append to self._last_mutation_events
         - Log: "Variant promoted to live"
      Return list of mutation events.

    - `def get_learning_status(self) -> dict`:
      Return comprehensive status:
      ```python
      {
          "total_trades": self._total_trades,
          "trades_until_evolve": self._config.evolve_every_n_trades - self._trades_since_evolve,

          "shadow_variants": self._shadow.get_variant_status(),

          "rule_status": self._retirement.get_rule_status(),

          "recent_mutations": self._last_mutation_events[-10:],
      }
      ```

    - `def get_retirement_tracker(self) -> RuleRetirementTracker`: Return for status queries.

       ```python
       def _update_engine_state(self) -> None:
           """Atomically update shared state snapshot for dashboard consumption."""
           s = self._engine_state
           # Regime from last signal
           if self._last_signals:
               chaos_signal = next((sig for sig in self._last_signals if sig.module_name == "chaos"), None)
               if chaos_signal and chaos_signal.regime:
                   s.regime = chaos_signal.regime
                   s.regime_confidence = chaos_signal.confidence
           # Signal confidences/directions
           s.signal_confidences = {sig.module_name: sig.confidence for sig in (self._last_signals or [])}
           s.signal_directions = {sig.module_name: sig.direction for sig in (self._last_signals or [])}
           # Fusion score
           if hasattr(self, '_last_fusion_result') and self._last_fusion_result:
               s.fusion_score = self._last_fusion_result.composite_score
           # Price and spread from latest tick
           if self._tick_buffer and len(self._tick_buffer) > 0:
               arrays = self._tick_buffer.as_arrays()
               s.current_price = float(arrays["bid"][-1])
               s.spread = float(arrays["spread"][-1])
           # Equity and P&L
           s.equity = self._current_equity if hasattr(self, '_current_equity') else 0.0
           # Breaker status
           if self._breakers:
               snapshot = self._breakers.get_snapshot()
               s.breaker_status = {"daily_drawdown": snapshot.daily_drawdown_state.value if hasattr(snapshot, 'daily_drawdown_state') else "unknown"}
           # Connection status
           s.is_connected = self._connected if hasattr(self, '_connected') else False
           # Kill switch
           if self._kill_switch:
               s.is_killed = self._kill_switch.is_killed if hasattr(self._kill_switch, 'is_killed') else False
           # Order flow from last flow signal
           if self._last_signals:
               flow_sig = next((sig for sig in self._last_signals if sig.module_name == "flow"), None)
               if flow_sig and flow_sig.metadata:
                   s.volume_delta = flow_sig.metadata.get("volume_delta", 0.0)
                   s.bid_pressure = flow_sig.metadata.get("bid_pressure", 0.0)
                   s.ask_pressure = flow_sig.metadata.get("ask_pressure", 0.0)
           # Recent trades from trade_logger
           if self._trade_logger:
               s.recent_trades = self._trade_logger.get_recent_trades(20)
           # Learning mutations
           if self._learning_loop:
               status = self._learning_loop.get_learning_status()
               s.recent_mutations = status.get("recent_mutations", [])
       ```
