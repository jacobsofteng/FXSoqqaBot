    **Class EvolutionManager:**
    - `__init__(self, config: LearningConfig)`:
      Create DEAP Toolbox. Register:
      - `toolbox.register("attr_float", _random_within_bounds)` -- generates random float for each param within its bounds
      - `toolbox.register("individual", tools.initIterate, creator.Individual, _create_random_individual)`
      - `toolbox.register("population", tools.initRepeat, list, toolbox.individual)`
      - `toolbox.register("mate", tools.cxBlend, alpha=0.5)`
      - `toolbox.register("mutate", tools.mutGaussian, mu=0, sigma=0.1, indpb=0.2)`
      - `toolbox.register("select", tools.selTournament, tournsize=config.ga_tournament_size)`
      Initialize population: `self._population = toolbox.population(n=config.ga_population_size)`
      Track generation count, best individual history.

    - `_create_random_individual() -> list[float]`: Returns list of random values, one per param, each within its PARAM_BOUNDS.

    - `_clamp_individual(individual) -> None`: Post-mutation, clamp each gene to its PARAM_BOUNDS. Called after mutate to ensure bounds.

    - `_phase_aware_fitness(self, individual: list, trades: list[dict], equity: float) -> tuple[float,]`:
      Convert individual to param dict via `individual_to_params()`.
      Compute metrics from trades list:
      - `_compute_profit_factor(trades)`: sum of winning pnl / abs(sum of losing pnl). If no losses, return 10.0 cap. If no wins, return 0.0.
      - `_compute_sharpe_ratio(trades)`: mean(pnl) / std(pnl) if std > 0, else 0.0. Annualize by sqrt(252).
      - `_compute_max_drawdown_penalty(trades)`: max peak-to-trough equity drawdown as fraction, return as positive penalty.

      Phase-aware selection per D-14:
      ```python
      if equity < 100:   # Aggressive
          return (_compute_profit_factor(trades),)
      elif equity < 300: # Selective
          return (_compute_sharpe_ratio(trades),)
      else:              # Conservative
          sharpe = _compute_sharpe_ratio(trades)
          dd_penalty = _compute_max_drawdown_penalty(trades)
          return (sharpe - dd_penalty,)
      ```

    - `run_generation(self, trades: list[dict], equity: float) -> dict`:
      Use `algorithms.varAnd()` to produce offspring. Evaluate fitness for each. Clamp to bounds. Select next generation. Increment generation counter. Return dict with generation number, best fitness, best params. Log with structlog.

    - `individual_to_params(self, individual: list) -> dict[str, float]`:
      Zip PARAM_NAMES with individual values to create param dict.

    - `get_best_individual(self) -> tuple[list[float], float]`:
      Return the individual with highest fitness from current population, plus its fitness value.

    - `get_state(self) -> dict`: Serialize population + generation for persistence.
    - `load_state(self, state: dict) -> None`: Restore from serialized state.

    Create `tests/test_evolution.py` with comprehensive tests. Use small population (size=5) for fast tests. Mock trade data with known P&L values to verify fitness computations.

    1. **src/fxsoqqabot/learning/analyzer.py**: Create SignalAnalyzer per D-20/LEARN-05.

       Class `SignalAnalyzer`:
       - `__init__(self)`: Initialize with structlog logger.

       - `analyze_combinations(self, trades: list[dict]) -> list[dict]`:
         For each trade, determine which signal modules had confidence >= 0.4 (active signal). Compute all observed 2-module and 3-module combinations. For each combination, calculate: win_rate, trade_count, avg_pnl. Return sorted by win_rate descending. Each result dict: `{"combination": ("chaos", "flow"), "win_rate": 0.74, "trade_count": 42, "avg_pnl": 1.2, "strength": "strong" if win_rate > 0.7 else "moderate" if win_rate > 0.5 else "weak"}`.

       - `analyze_regime_performance(self, trades: list[dict]) -> dict[str, dict]`:
         Group trades by regime field. For each regime: compute win_rate, trade_count, avg_pnl, profit_factor. Return dict keyed by regime name.

       - `identify_degrading_rules(self, trades: list[dict], window_size: int = 50) -> list[dict]`:
         Compare win rate of last `window_size` trades vs all trades. Rules where recent win_rate is more than 15 percentage points below overall are "degrading". Return list of {"rule": param_name, "overall_win_rate": float, "recent_win_rate": float, "delta": float}.

       - `get_summary(self, trades: list[dict]) -> dict`:
         Returns complete analysis: best_combinations, worst_combinations, regime_performance, degrading_rules.

    2. **src/fxsoqqabot/learning/retirement.py**: Create RuleRetirementTracker per D-19, mirroring the AdaptiveWeightTracker EMA pattern from `signals/fusion/weights.py`.

       Class `RuleRetirementTracker`:
       - `__init__(self, rule_names: list[str], alpha: float = 0.1, min_trades: int = 50, retirement_threshold: float = 0.3)`:
         Initialize EMA scores to 0.5 for each rule. Track trade counts. Maintain `active_rules: set[str]`, `cooldown_pool: dict[str, dict]` (stores rule name -> {"retired_at": datetime, "ema_at_retirement": float, "params_snapshot": dict}).

       - `record_outcome(self, rule_name: str, success: bool) -> None`:
         EMA update: `new_ema = alpha * (1.0 if success else 0.0) + (1.0 - alpha) * old_ema`. Increment trade count. If trade_count >= min_trades and ema < retirement_threshold and rule is active: retire it.

       - `_retire_rule(self, rule_name: str) -> None`:
         Move rule from active_rules to cooldown_pool with timestamp and EMA snapshot. Log with structlog: "Rule retired", rule=rule_name, ema=score.

       - `reactivate_rule(self, rule_name: str, mutated_params: dict | None = None) -> None`:
         Remove from cooldown_pool, add back to active_rules, reset EMA to 0.5, reset trade_count to 0.

       - `get_rule_status(self) -> dict[str, dict]`:
         For each rule: {"ema": float, "trade_count": int, "status": "active"|"retired"|"cooldown"}.

       - `get_active_rules(self) -> list[str]`: Return list of currently active rule names.
       - `get_retired_rules(self) -> list[dict]`: Return cooldown pool entries.
       - `get_state(self) -> dict`: Serialize for persistence.
       - `load_state(self, state: dict) -> None`: Restore from serialized state.
