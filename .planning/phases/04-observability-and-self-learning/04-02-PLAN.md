    3. **src/fxsoqqabot/dashboard/tui/widgets.py**: Create pure formatting functions that take TradingEngineState fields and return Rich markup strings. These are testable without Textual:

       - `format_regime_panel(regime: RegimeState, confidence: float) -> str`:
         Maps regime to color per D-02/UI-SPEC: TRENDING_UP/TRENDING_DOWN -> "green", RANGING -> "yellow", HIGH_CHAOS/PRE_BIFURCATION -> "red". Returns `f"[bold {color}]REGIME: {regime.value.upper()}[/] ({confidence:.0%})"`. If disconnected, return `"[bold red]MT5 DISCONNECTED[/] -- Reconnecting..."`.

       - `format_signals_panel(confidences: dict[str, float], directions: dict[str, float]) -> str`:
         For each module (chaos, flow, timing): direction arrow is "^" if direction > 0, "v" if < 0, "-" if == 0. Color by confidence tier: >= 0.70 green, 0.40-0.69 yellow, < 0.40 red. Format each line as `f"[{color}]{name.title():8s}: {arrow} {conf:.0%}[/]"`. Join with newlines.

       - `format_position_panel(position: dict | None) -> str`:
         If None: return `"No open position"`. Otherwise: `f"{pos['action'].upper()} {pos['lots']:.2f} @ {pos['price']:.2f} | P&L: {pnl_str} | SL: {pos['sl']:.2f}"` where pnl_str uses green for positive, red for negative.

       - `format_risk_panel(breaker_status: dict[str, str], is_killed: bool) -> str`:
         If is_killed: return `"[bold red on white]KILLED[/]"`. Extract daily_drawdown from breaker_status. Format: `f"Daily DD: {dd} | Breakers: {status}"`. If any breaker is "TRIPPED": `"[bold red]HALTED ({reason})[/]"`.

       - `format_order_flow(volume_delta: float, bid_pressure: float, ask_pressure: float) -> str`:
         Returns `f"Delta: {volume_delta:+.0f} | Bid: {bid_pressure:.0%} Ask: {ask_pressure:.0%}"`.

       - `format_stats_panel(trade_count: int, win_rate: float, daily_pnl: float, equity: float) -> str`:
         Returns `f"Trades: {trade_count} | Win: {win_rate:.0%} | P&L: {pnl_str} | Equity: ${equity:.2f}"`.

       - `format_trade_row(trade: dict) -> list[str]`:
         Returns a list for DataTable columns: [time, action, lots, entry, exit, pnl, regime]. Time formatted as HH:MM from trade["timestamp"].

       - `is_mutation_event(trade: dict) -> bool`:
         Returns True if trade has key "mutation" set to True.

       - `format_mutation_row(mutation: dict) -> str`:
         Returns `f"[bold magenta][MUTATED] {mutation['param']}: {mutation['old']} -> {mutation['new']} ({mutation['reason']})[/]"` per D-03.

    Class `FXSoqqaBotTUI(App)`:
    - `TITLE = "FXSoqqaBot - XAUUSD Scalper"` (UI-SPEC Header copy)
    - `CSS_PATH = Path(__file__).parent / "styles.tcss"` (relative to app.py)
    - `BINDINGS` list:
      - `Binding("q", "quit_app", "Quit")` -- quit TUI only, engine keeps running
      - `Binding("k", "kill_all", "Kill")` -- trigger kill switch
      - `Binding("p", "toggle_pause", "Pause")` -- toggle trading pause

    - `__init__(self, state: TradingEngineState, kill_callback=None, pause_callback=None)`:
      Store state reference, kill_callback, pause_callback. Call `super().__init__()`.

    - `compose(self) -> ComposeResult`:
      Match the UI-SPEC layout exactly:
      ```
      yield Header()
      with Horizontal():
          with Vertical(id="left-panel"):
              yield Static("", id="regime-panel", classes="panel")
              yield Static("", id="signals-panel", classes="panel")
              yield Static("", id="order-flow-panel", classes="panel")
          with Vertical(id="center-panel"):
              yield Static("", id="position-panel", classes="panel")
              yield Static("", id="risk-panel", classes="panel")
              yield DataTable(id="trades-table", classes="panel")
          with Vertical(id="right-panel"):
              yield Static("", id="stats-panel", classes="panel")
              yield Sparkline([], id="equity-spark", classes="panel")
              yield Button("KILL ALL POSITIONS", id="kill-btn", variant="error")
      yield Footer()
      ```

    - `on_mount(self) -> None`:
      - `self.set_interval(1.0, self._refresh_all)` per D-05
      - Initialize the DataTable columns: Time, Action, Lots, Entry, Exit, P&L, Regime (per UI-SPEC Trades table columns)
      - `self.query_one("#trades-table", DataTable).cursor_type = "row"`

    - `_refresh_all(self) -> None`:
      Read from `self._state` and update each panel using the formatting functions from widgets.py:

      - `self.query_one("#signals-panel", Static).update(format_signals_panel(...))`
      - `self.query_one("#order-flow-panel", Static).update(format_order_flow(...))`
      - `self.query_one("#position-panel", Static).update(format_position_panel(...))`
      - `self.query_one("#risk-panel", Static).update(format_risk_panel(...))`
      - `self.query_one("#stats-panel", Static).update(format_stats_panel(...))`
      - Update Sparkline data: `self.query_one("#equity-spark", Sparkline).data = self._state.equity_history[-50:]`
      - For trades table: clear and repopulate from `self._state.recent_trades` (last 20). For each trade, if `is_mutation_event(trade)`, add the row and style it. Otherwise add normal row via `format_trade_row(trade)`.
      - Also process `self._state.recent_mutations` and render inline per D-03.

    - `action_quit_app(self) -> None`: Log "TUI exiting, engine continues", call `self.exit()`.

    - `action_kill_all(self) -> None`: If kill_callback: call it. Log "Kill switch activated from TUI".

    - `action_toggle_pause(self) -> None`: If pause_callback: call it. Toggle subtitle between "[PAUSED]" and "".

    - `on_button_pressed(self, event: Button.Pressed) -> None`: If event.button.id == "kill-btn": call `self.action_kill_all()`.
