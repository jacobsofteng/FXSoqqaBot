<objective>
Fix all four dashboard wiring bugs so TUI and web dashboards display accurate live data and the pause command stops trading.

Purpose: The dashboards were built in Phase 4 but the engine never writes correct values to the shared TradingEngineState -- equity is always 0, connection is always False, is_killed is a truthy coroutine, equity_history is empty, module weights are empty, breaker status silently fails, and pause is set but never checked. This plan fixes all wiring bugs in a single pass.

Output: Four production files modified with all bugs fixed. No new files created.
</objective>
