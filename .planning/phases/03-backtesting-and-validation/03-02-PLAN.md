    **__init__(self, config: BacktestConfig)**:

    **parse_histdata_csv(self, filepath: Path) -> pd.DataFrame**:
    Per D-01 and Research Pattern 4:
    - Use `pd.read_csv(filepath, sep=";", header=None, names=["datetime_str", "open", "high", "low", "close", "volume"])`
    - dtype enforcement: open/high/low/close as float64, volume as int64
    - Parse datetime: `pd.to_datetime(df["datetime_str"], format="%Y%m%d %H%M%S")`
    - Convert EST to UTC: add `timedelta(hours=5)` (NO DST per histdata.com spec -- Research Pitfall 2)
    - Compute `time` column: Unix timestamp in seconds (int64) from UTC datetime
    - Add `year` and `month` columns from UTC datetime for Parquet partitioning
    - Drop intermediate datetime_str and datetime_est columns
    - Return DataFrame with columns: datetime_utc, time, open, high, low, close, volume, year, month

    **validate_bar_data(self, df: pd.DataFrame) -> tuple[pd.DataFrame, dict]**:
    Per D-03:
    1. Check monotonic timestamps: `df["time"].is_monotonic_increasing`. If not, drop duplicates (keep="first"), sort by time.
    2. Detect gaps: compute `np.diff(df["time"].values)`. Expected interval = 60 seconds for M1.
       - Small gaps (1.5x to 5x expected): count, forward-fill interpolation (create synthetic bars with close=previous close, high=low=open=close, volume=0)
       - Large gaps (>5x expected): log in quality report, do NOT fill (weekends, holidays, outages)
    3. Filter extreme bars: compute `bar_range = high - low`. Remove bars where `bar_range > mean_range * 10`. Log count.
    4. Flag zero-volume bars: count and report but do NOT remove (common during low liquidity per Research).
    5. Build quality_report dict: `{"original_rows": int, "final_rows": int, "issues": list[str], "date_range": (min_utc, max_utc), "duplicates_removed": int, "gaps_interpolated": int, "extreme_bars_removed": int, "zero_volume_bars": int, "large_gaps": int}`
    6. Return (cleaned_df, quality_report)

    **convert_to_parquet(self, df: pd.DataFrame, output_dir: Path | None = None) -> Path**:
    Per D-04:
    - Use DuckDB to write Parquet with partitioning: `COPY ... TO '{output_dir}' (FORMAT PARQUET, PARTITION_BY (year, month))`
    - Output dir defaults to `config.parquet_dir`
    - Return the output directory path

    **ingest_all(self, csv_dir: Path | None = None) -> dict**:
    - Glob all `*.csv` files in csv_dir (defaults to config.histdata_dir)
    - For each CSV: parse -> validate -> accumulate into one large DataFrame
    - Convert combined DataFrame to Parquet
    - Return combined quality report

    **load_bars(self, start_time: int, end_time: int) -> pd.DataFrame**:
    - Query Parquet via DuckDB: `SELECT time, open, high, low, close, volume FROM read_parquet('{parquet_dir}/**/*.parquet') WHERE time >= {start_time} AND time < {end_time} ORDER BY time`
    - Uses start-inclusive, end-exclusive window boundaries (Phase 1 convention)
    - Return DataFrame

    **get_time_range(self) -> tuple[int, int]**:
    - Query min/max time from Parquet: `SELECT MIN(time), MAX(time) FROM read_parquet(...)`
    - Return (min_time, max_time)
